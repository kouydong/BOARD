# 게사판 만들기

## 게시판 기본 기능 

### 환경
- RDB : CLOUD POSTGRES
- DB CONNECTION : JPA
- Template Engine : Thymeleaf

#### 참고사항
 - 댓글(CRUD) 테이블(Comment) Entity 까지는 생성하였으나, 구현하지 않음
 - 회원정보는 MBL Token을 통해서 조회 할 수 있도록 수정 필요
 - RestAPI 형태가 아니므로 GET, POST만 사용
 - 업로드 기능 구현하지 않음
